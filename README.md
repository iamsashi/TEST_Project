# 🎯 Face Detection Project (P1 & P2)

## 📌 Overview

This project demonstrates real-time face detection using **MTCNN (Multi-task Cascaded Convolutional Neural Network)** and **OpenCV**.

It is divided into two modules:

* **P1** → Basic real-time face detection using webcam
* **P2** → Extended/advanced implementation (can include optimization, enhancements, or additional features)

---

## 🚀 Features

* 🎥 Real-time webcam-based face detection
* 🧠 Deep learning-based detection using MTCNN
* ⚡ Fast processing with OpenCV integration
* 📦 Modular structure (P1 & P2 separated)
* 🖥️ Easy to run and extend

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/iamsashi/TEST_Project.git
cd TEST_Project
```

---

### 2. Create virtual environment (recommended)

```bash
python -m venv mtcnn_env
.\mtcnn_env\Scripts\Activate.ps1   # Windows
```

---

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ How to Run

### ▶ Run P1 (Basic Face Detection)

```bash
python P1/P1.py
```

---

### ▶ Run P2 (Advanced Module)

```bash
python P2/P2.py
```
